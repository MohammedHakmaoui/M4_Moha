¿Qué es un Repositori?

Un repositori, en el context de desenvolupament de programari, és un lloc on s'emmagatzemen i gestionen fitxers de codi font i altres recursos relacionats. Els repositoris són essencials per al control de versions i la col·laboració en projectes de desenvolupament. Els sistemes de control de versions com Git permeten als equips de desenvolupament rastrejar canvis, col·laborar i mantenir un historial complet de la seva feina.

Diferencia entre un pagina Web Estatica i una Dinamica

Web Estática Un lloc web estàtic mostra contingut fixe sense intervenció. El contingut es crea prèviament i es guarda en fitxers HTML, CSS i JavaScript. Els visitants veuran la mateixa informació cada cop que accedeixin al lloc. Els llocs estàtics són més simples i ràpids d'allotjar, però no permeten interaccions en temps real ni actualitzacions automàtiques.

Web Dinámica Un lloc web dinàmic genera contingut de manera dinàmica en resposta a les sol·licituds dels usuaris. Utilitza bases de dades i llenguatges de programació com ara PHP, Python o JavaScript amb Node.js, per crear pàgines web personalitzades. Això permet la interacció en temps real, la personalització i lactualització automàtica de contingut. Els llocs web dinàmics són ideals per a aplicacions web complexes, com ara xarxes socials i botigues en línia.
